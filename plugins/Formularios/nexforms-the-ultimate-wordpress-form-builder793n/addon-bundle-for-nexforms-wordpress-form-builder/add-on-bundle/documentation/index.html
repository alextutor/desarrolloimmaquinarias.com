Please note: This plugin is dependant on NEX-Forms - The Ultimate WordPress Form Builder version 6.1.3 or higher

Installation

VIA WordPress

  1. Extract each add-on from the main .zip folder (as downloaded)
  3. Log into your WordPress admin interface
  4. After login click on  Plugins from the left  hand menu
  5. Click on “Add new”
  6. Click on “Upload”
  7. Browse to the directory you extracted the add-ons to.
  9. Select the add-on and click “Install Now”
  10. After Wordpress has finished unpacking the file click on “Activate Plugin”
  11. After the add-on has been activated go to NEX-Forms
     
  12. Repeat these steps for every add-on
     

VIA FTP

  1. Extract each add-on from the main .zip folder (as downloaded)
  3. Unzip each add-on
  4. Open your FTP client
  5. Browse to /your_wp_instalation/wp-content/plugins/ server
  6. Upload Nex-Forms add-ons into this directory
  7. Go to your wordpress admin panel
  8. Go to plugins and activate each add-on
